# Chatbot Baseado em PDFs 📄🤖

## Descrição do Projeto
Este projeto cria um **chatbot interativo** que responde perguntas com base no conteúdo de arquivos PDF carregados pelo usuário. Ideal para estudantes ou pesquisadores que precisam extrair informações de múltiplos documentos.

## Objetivos
- Carregar PDFs relevantes.
- Criar um **sistema de busca vetorial** com embeddings.
- Gerar respostas fundamentadas no conteúdo dos PDFs.
- Desenvolver um chat interativo para perguntas e respostas.

## Estrutura do Projeto
- `inputs/` → Exemplo de texto para teste.
- `data/pdfs/` → PDFs que alimentam o chatbot.
- `notebooks/` → Testes e análises exploratórias.
- `src/` → Scripts para carregar PDFs, gerar embeddings e rodar o chat.
- `requirements.txt` → Dependências do projeto.
- `README.md` → Documentação.
